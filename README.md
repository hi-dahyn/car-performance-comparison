# 브랜드 별 자동차 성능 비교


### 프로젝트 개요:

이 프로젝트는 Spark를 사용하여 자동차 데이터를 효율적으로 로드하고, 변환하며, 분석하는 것을 목표로 합니다. 주된 목적은 브랜드별로 자동차의 성능을 비교하고, 특정 브랜드가 다른 브랜드에 비해 얼마나 강력한 엔진을 가지고 있는지 평가하기 위함입니다.

### 주요 과정:

- **RDD 연산 사용**: 다양한 RDD 연산(map, filter, flatMap, reduce, distinct 등)을 사용하여 데이터를 변환하고, 필터링하고, 분석하였습니다. 각 연산의 목적과 사용 방법을 명확히 하여 데이터를 효율적으로 처리하였습니다.
- **데이터 정제 및 변환**: 사용자 정의 함수를 사용하여 데이터를 정제하고 변환하였습니다. 이를 통해 데이터의 일관성과 가독성을 높였습니다.
- **집계 및 분석**: `reduce`와 `reduceByKey` 연산을 사용하여 데이터를 집계하고 분석하였습니다. 이를 통해 데이터의 총합, 평균 등을 계산하였습니다.

### 결과:

이 프로젝트를 통해 Spark를 활용한 대규모 데이터 처리와 분석의 효율성을 체험할 수 있었습니다. 다양한 RDD 연산을 사용하여 데이터를 로드, 변환, 분석하는 과정을 통해 실무에서의 데이터 처리 능력을 향상시켰습니다. 이는 대규모 데이터를 다루는 모든 프로젝트에서 매우 유용한 기술임을 확인할 수 있었습니다.
